# 에이전트 / 기여자 안내

이 저장소에서 코딩·리뷰·자동화 에이전트가 작업할 때는 다음을 **반드시** 따른다.

1. **프로젝트 지침**: Cursor 규칙 **`.cursor/rules/rebac-engine-charter.mdc`** (`alwaysApply: true`) — 엔진의 PDP 역할, 모듈 경계, 패키지 네이밍, 데모 제거 상태 등.
2. **기능·우선순위**: **`docs/ENGINE-ROADMAP.md`** — Feature ID(F-01 …), Phase, PEP/PDP 경계, SPI 진화 원칙.

새 기능은 로드맵에 ID를 맞추거나 로드맵을 먼저 수정한 뒤 구현한다.
